# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please do not open a public GitHub issue.

Instead, report it privately by contacting me directly on Telegram:
@mork1e

Please include:
- A description of the issue
- Steps to reproduce
- Affected files / code references
- Potential impact
- Suggested mitigation if known

I will do my best to acknowledge reports within 72 hours.

Thank you for helping improve the security of the faucet :)
