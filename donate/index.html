NewFree Bitcoins

Donate to the Faucet

This faucet does not use a centralized donation wallet. Donors send BTC directly to queued faucet participants, so before you can donate you need a wallet that stays under your control.

How It Works

1. Create a new donation wallet here, or import an existing mnemonic phrase.

2. Deposit BTC into that donation wallet. This is the wallet the faucet will use when it sends funds to approved participants.

3. When donating is enabled, your browser will use that donation wallet to construct and submit transactions directly to queued faucet requests.

4. Your mnemonic phrase is encrypted locally with your password, and each time you reload the page you unlock the donation wallet again with that password.

5. You stay in control. The wallet is self-custodial, there is no centralized faucet wallet, and you can stop donating at any time by locking the wallet or moving the funds elsewhere.

Why The Mnemonic Matters

The mnemonic phrase is the real backup. It controls the donation wallet and lets you recover the funds later in any compatible wallet.

The password only protects the local copy saved in this browser. If you lose the mnemonic phrase, the password alone is not enough to recover the wallet somewhere else.

Want to run a donation wallet on your VPS?

Install the donor CLI in one command:

curl -fsSL https://raw.githubusercontent.com/newfreebitcoins/newfreebitcoins-mono/main/install.sh | sh
